# План действий
## 1) Датасет

1.1) собрать на диске файлы из исходников RNNoise + корпус языка + шум (где взять?)

1.2) разбивка на обучающую и валидационную выборки

## 2) Tuning

2.1) Запустить обучалку на этих данных, записать метрики, прогнать на валидационной для сравнения

2.2) Вытащить всё из колаба в локальное хранилище, протестить локально

## 3) Интерфейс

3.1) Интерфейс для офлайн (не потоковой обработки) записей, простенький

3.2) API-сервис, отвечающий в онлайне

3.3) Опрос сервиса системным микрофоном, подключение к системному микрофону

## 4) Докеризация

4.1) Запаковать всё, протестировать в контейнере
